# TypeScript React Boilerplate

[![Build Status](https://travis-ci.org/xcatliu/ts-react.svg?branch=master)](https://travis-ci.org/xcatliu/ts-react)

This is a boilerplate with tech below:

- TypeScript
- React
- Redux
- React Router

## Getting Started

```bash
npm i
npm start

# Build to docs
npm run build
```

Open http://localhost:8080/
